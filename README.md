## DB

- Os dados do banco estarão presentes no arquivo `.env`:
```
    DB_CONNECTION=mysql
    DB_HOST=*localhost*
    DB_PORT=*porta*
    DB_DATABASE=*nome*
    DB_USERNAME=*usuario*
    DB_PASSWORD=*senha*
```
